HTML layouts create an individual space for every part of the web page.

HTML Layouts

  1. Header: defines a header for the document/section.
  2. Nav: defines a container for a navigation links.
  3. Section: defines a section in a document.
  4. Article: defines an independent or self-contained article.
  5. Aside: defines content aside from the content (like a sidebar).
  6. Footer: defines a footer for the document section.
  7. Details: defines additional details.
  8. Summary: defines a summary for the details element.
